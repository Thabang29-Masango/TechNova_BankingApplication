# TechNova Banking Application

## Overview

This project was developed as part of the TechNova Java Programming Internship.

It is a console-based Java banking application that demonstrates Object-Oriented Programming, exception handling, and user authentication.

## Features

- PIN authentication
- Three login attempts
- Account locking after three failed attempts
- Deposit money
- Withdraw money
- Check account balance
- Invalid input handling
- Insufficient funds validation
- Encapsulation of account information

## Technologies Used

- Java
- Apache NetBeans
- Object-Oriented Programming

## Concepts Demonstrated

- Classes and objects
- Constructors
- Encapsulation
- Methods
- Conditional statements
- Loops
- Exception handling
- Menu-driven applications

## How to Run

1. Download or clone the repository.
2. Open the project in Apache NetBeans.
3. Run `TechNova_BankingApplication.java`.
4. Enter the correct PIN to access the banking menu.
